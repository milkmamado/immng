# CRM 환자 정보 연동 Chrome 확장 프로그램

한국도움기술 CRM에서 환자 정보를 자동으로 추출하여 환자 관리 시스템으로 전송하는 Chrome 확장 프로그램입니다.

## 📦 설치 방법

### 1. 확장 프로그램 파일 다운로드
이 폴더의 모든 파일을 컴퓨터에 다운로드하세요.

### 2. 아이콘 파일 추가 (필수)
확장 프로그램에는 3가지 크기의 아이콘이 필요합니다:
- `icon16.png` (16x16 픽셀)
- `icon48.png` (48x48 픽셀)
- `icon128.png` (128x128 픽셀)

간단한 아이콘을 만들거나, 온라인 아이콘 생성기를 사용하세요.
또는 임시로 같은 이미지를 3개 크기로 저장해도 됩니다.

### 3. Chrome에 확장 프로그램 로드

1. **Chrome 브라우저 열기**

2. **확장 프로그램 관리 페이지로 이동**
   - 주소창에 `chrome://extensions/` 입력
   - 또는 메뉴 → 도구 더보기 → 확장 프로그램

3. **개발자 모드 활성화**
   - 우측 상단의 "개발자 모드" 토글 ON

4. **압축 해제된 확장 프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 버튼 클릭
   - chrome-extension 폴더 선택

5. **확장 프로그램 확인**
   - "CRM 환자 정보 연동" 확장 프로그램이 목록에 나타남
   - 브라우저 툴바에 아이콘이 표시됨

## 🎯 사용 방법

### 방법 1: 환자 정보 직접 추출

1. **CRM 로그인**
   - http://192.168.1.101/html/MEDI20/main.html 접속

2. **환자 조회**
   - 전화번호로 환자 검색
   - 환자명 더블클릭하여 상세정보 표시

3. **확장 프로그램 사용**
   - 브라우저 툴바의 확장 프로그램 아이콘 클릭
   - "환자 정보 가져오기" 버튼 클릭

4. **자동 등록**
   - 환자 관리 시스템이 새 탭으로 열림
   - 환자 정보가 자동으로 입력됨
   - "등록" 버튼만 클릭하면 완료!

### 방법 2: 환자 관리 시스템에서 시작 (향후 추가 예정)

1. 환자 관리 시스템에서 "조회" 버튼 클릭
2. 자동으로 CRM 창이 열리고 검색 실행
3. 환자 선택 후 확장 프로그램으로 정보 전송

## ⚠️ 문제 해결

### "확장 프로그램을 로드할 수 없습니다" 오류
- 모든 파일(manifest.json, content.js, popup.html 등)이 같은 폴더에 있는지 확인
- 아이콘 파일(icon16.png, icon48.png, icon128.png)이 있는지 확인

### "환자 정보를 찾을 수 없습니다" 메시지
- 환자를 더블클릭하여 상세정보가 표시되었는지 확인
- 상세정보가 완전히 로딩될 때까지 기다린 후 다시 시도

### 확장 프로그램 아이콘이 안 보임
- Chrome 툴바에서 퍼즐 아이콘 클릭
- "CRM 환자 정보 연동" 옆의 핀 아이콘 클릭하여 고정

### CRM 페이지에서 작동하지 않음
- 페이지 새로고침 (F5)
- 확장 프로그램 비활성화 후 다시 활성화
- Chrome 브라우저 재시작

## 🔄 업데이트

1. `chrome://extensions/` 페이지에서
2. "CRM 환자 정보 연동" 카드의 새로고침 버튼 클릭
